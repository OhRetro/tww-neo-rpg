# Aprimorar Corpo

O Corpo do Usuario é Aprimorado para o Combate.

Rode 1d4 para Determinar o Melhoramento de Proteção.

Ação de Movimento.
Alcance: Pessoal
Duração: Cena
Custo de PE: 3 PE

Upgrade I:
Pode Aprimorar com uma Melhoria Escolhida (+2 PE)

Upgrade II:
Capaz de Aprimorar Proteção no Corpo (+2 PE)
